Nízky príjem soli môže byť zdraviu škodlivý

Diéta s nízkym obsahom soli môže byť dokonca zdraviu škodlivá.

Negatívne účinky nedostatku soli na zdravie:

  1. Obmedzenie soli je spájané so zvýšením zlého LDL cholesterolu a triglyceridov
    2. => LDL cholesterol (zlý cholesterol)
    => viac o cholesterole
    =>niekoľko klinických štúdií potvrdzuje, že príjem menej ako 3000 mg sodíka denne zvyšuje riziko smrti na infarkt myokardu.
  2. Srdcové zlyhanie
    3. => u niektorých pacientov na ktorých sa viedol výskum, sa ukázalo, že zníženie príjmu soli viedlo k zvýšeniu rizika smrti u pacientov so zlyhávajúcim srdcom.

Šokujúce výsledky

  1. ukázalo sa totiž, že ľudia so zlyhávajúcim srdcom, ktorí znížili príjem soli, mali o 160% vyššie riziko smrti než tí, ktorí príjem soli nezníži.
  2. Inzulínová rezistencia
    3. => niektoré klinické štúdie preukázali, že diéta s nízkym obsahom soli môže viesť k zvýšeniu inzulínovej rezistencie.
    => viac o inzulinovej rezistencii
  3. Diabetes 2. typu
    4. =>zníženie príjmu sodíka môže viest k zvýšeniu rizika smrti.
